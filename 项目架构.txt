用户输入："上个月华南区销售额为什么下降了？"
        ↓
┌─────────────────────────────────────┐
│  Agent 1: 需求分析师 (Planner)       │
│  职责：理解模糊的业务问题，拆解成     │
│  可执行的子任务清单                   │
│  输出：["查询上个月华南区各品类销售额", │
│         "对比去年同期数据",            │
│         "查询华南区退货率变化"]        │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│  Agent 2: SQL工程师 (Coder)          │
│  职责：根据Schema和子任务，逐个生成   │
│  正确的SQL语句                       │
│  输出：SELECT product_category,       │
│  SUM(amount) FROM orders WHERE...     │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│  Agent 3: 安全检查员 (Guard)          │
│  职责：检查SQL安全性 + 执行 + 捕获结果 │
│  如果SQL报错，把错误信息返回给Agent 2  │
│  让其自我修正                        │
└──────────────┬──────────────────────┘
               ↓
┌─────────────────────────────────────┐
│  Agent 4: 报告分析师 (Reporter)       │
│  职责：接收所有查询结果，综合分析，   │
│  用自然语言生成诊断报告               │
│  输出："上月华南区销售额下降12%，      │
│  主要原因是电子品类退货率上升3倍..."   │
└─────────────────────────────────────┘
